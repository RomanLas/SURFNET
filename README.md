# SURFNET
Program for visualizing molecular surfaces, cavities and intermolecular interaction interfaces.

Also included is a new version called speedfill which uses a slightly improved algorithm but doesn't output all the same file formats.

To install, download the following files:

<UL>
  <LI><B>surfnet.tar.gz</B> - SURFNET executables and source code
  <LI><B>docs.tar.gz</B> - documentation
</UL>

Extract all the files from both archives. Then follow the istallation instructions in <B>install.html</B> (from the extracted <B>docs.tar.gz</B> file).
